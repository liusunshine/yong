# Yong(使用)


[caniuse.com](caniuse.com) in your command line.


---

## Install

```bash
$ npm install yong -g
```

## Usage

I set up a simple server with nodejs and [jae](http://jae.jd.com/).

The data is fetched from [ghithub => caniuse](https://github.com/Fyrd/caniuse).

Translate one word.

```bash
$ yong border-radius
```
